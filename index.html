FLOWFREELANCE

Politique de confidentialité

Dernière mise à jour : 3 avril 2026
FlowFreelance ne collecte aucune donnée sur ses serveurs. Toutes vos données financières restent sur votre appareil.

1. Présentation

FlowFreelance est une application mobile de gestion financière destinée aux freelances français. Elle est éditée par Charbih.

Cette politique explique comment l'application gère vos données personnelles.

2. Données collectées

2.1 Données saisies par l'utilisateur

FlowFreelance collecte uniquement les données que vous saisissez volontairement :

  • Votre prénom (optionnel, pour personnaliser l'affichage)
  • Votre statut juridique (micro-entreprise, EURL, etc.)
  • Votre TJM et objectifs financiers
  • Vos missions et revenus mensuels
  • Vos charges fixes mensuelles

Ces données sont stockées exclusivement sur votre appareil via AsyncStorage. Elles ne sont pas transmises à nos serveurs.

2.2 Données collectées automatiquement

L'application ne collecte aucune donnée de navigation, aucun identifiant publicitaire, et n'utilise aucun tracker analytique.

2.3 Données liées aux achats in-app

Si vous souscrivez à FlowFreelance Premium, la transaction est gérée par Google Play ou Apple App Store. Nous utilisons RevenueCat pour vérifier l'état de votre abonnement. RevenueCat peut traiter un identifiant d'achat anonyme. Aucune donnée bancaire ne transite par notre application.

Politique de RevenueCat : revenuecat.com/privacy

3. Utilisation des données

Les données saisies sont utilisées uniquement pour :

  • Calculer vos indicateurs financiers (net estimé, URSSAF, TVA)
  • Générer des alertes et recommandations personnalisées
  • Produire vos exports comptables (CSV)

Ces calculs sont effectués entièrement en local sur votre appareil.

4. Partage des données

Nous ne vendons ni ne partageons vos données personnelles, sauf :

  • RevenueCat : gestion des abonnements (identifiant anonyme uniquement)
  • Obligation légale : si la loi nous y oblige

5. Conservation des données

Vos données sont conservées sur votre appareil tant que vous utilisez l'application. La désinstallation supprime l'ensemble de vos données locales.

Vous pouvez supprimer vos données à tout moment via Profil → Réinitialiser le profil.

6. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : toutes vos données sont visibles dans l'application
  • Rectification : modifiez vos données à tout moment
  • Effacement : Profil → Réinitialiser le profil
  • Portabilité : export CSV disponible dans Profil

Pour toute demande : contact@flowfreelance.app

7. Mineurs

FlowFreelance est destinée aux professionnels indépendants adultes. L'application n'est pas destinée aux mineurs de moins de 18 ans.

8. Modifications

Nous nous réservons le droit de modifier cette politique à tout moment. La date de mise à jour est indiquée en haut de ce document.
